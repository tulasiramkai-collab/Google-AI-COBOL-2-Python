IDENTIFICATION DIVISION.
PROGRAM-ID. AGE-CALCULATOR.

DATA DIVISION.
WORKING-STORAGE SECTION.
01 WS-BIRTH-DATE.
   05 WS-BIRTH-YEAR   PIC 9(4).
   05 WS-BIRTH-MONTH  PIC 9(2).
   05 WS-BIRTH-DAY    PIC 9(2).

01 WS-CURRENT-DATE.
   05 WS-CURRENT-YEAR   PIC 9(4).
   05 WS-CURRENT-MONTH  PIC 9(2).
   05 WS-CURRENT-DAY    PIC 9(2).

01 WS-AGE            PIC 9(3).

PROCEDURE DIVISION.
MAIN-LOGIC.
    DISPLAY "Enter Birth Date (YYYYMMDD): ".
    ACCEPT WS-BIRTH-DATE.

    DISPLAY "Enter Current Date (YYYYMMDD): ".
    ACCEPT WS-CURRENT-DATE.

    PERFORM CALCULATE-AGE.

    DISPLAY "The person's age is: " WS-AGE.

    STOP RUN.

CALCULATE-AGE.
    COMPUTE WS-AGE = WS-CURRENT-YEAR - WS-BIRTH-YEAR.

    IF WS-CURRENT-MONTH < WS-BIRTH-MONTH
        SUBTRACT 1 FROM WS-AGE
    ELSE IF WS-CURRENT-MONTH = WS-BIRTH-MONTH
        IF WS-CURRENT-DAY < WS-BIRTH-DAY
            SUBTRACT 1 FROM WS-AGE
        END-IF
    END-IF.
